# 刘炯杰

这是一个可直接用于 GitHub Pages 的个人学术主页。页面内容根据个人简历整理，重点展示个人研究方向、教育与工作经历、主持项目、奖励、论文、专利和联系方式。

## 文件结构

- `index.html`：网页正文
- `styles.css`：页面样式
- `assets/research-graphic.svg`：首页研究图形摘要

## 后续可补充

- Google Scholar 链接
- ORCID 链接
- 课题组主页链接
- 完整论文 DOI / PDF 链接
- 会议报告、口头报告、墙报展示等条目

## 发布到 GitHub Pages

1. 在 GitHub 新建仓库，例如 `username.github.io`。
2. 上传本文件夹中的全部内容，包括 `index.html`、`styles.css` 和 `assets` 文件夹。
3. 打开仓库的 `Settings` -> `Pages`，选择从 `main` 分支发布。
4. 等待一两分钟后访问 `https://username.github.io`。

如果使用项目仓库发布，例如仓库名为 `academic-homepage`，发布地址通常是 `https://username.github.io/academic-homepage/`。
